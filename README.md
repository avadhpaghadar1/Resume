# Resume Portfolio

Personal portfolio website built with React.

## Setup

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Customize content

- Update your details and sections in `src/portfolio.js`.

## Deploy (GitHub Pages)

- This project uses hash-based routing for GitHub Pages compatibility.
- Run:

```bash
npm run deploy
```

Notes:

- Set your repo to publish from the `gh-pages` branch in GitHub settings.
- Update your social links in `src/portfolio.js` (`greeting.githubProfile`, `greeting.linkedinProfile`).
